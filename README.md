## 🎯 Introduction

**Edemy** is a **comprehensive online learning platform** that connects **educators with students** through interactive courses.  
Built with **modern web technologies**, it provides:  

- 🎓 Dual dashboards for educators & students  
- 🔒 Secure authentication & payments  
- 🎥 Video streaming with progress tracking  
- 📊 Analytics dashboard for insights  
- 🚀 A scalable, modular architecture  

---

## ⚙️ Tech Stack

💻 **Frontend:** React.js, Vite, TailwindCSS  
🛠️ **Backend:** Node.js, Express.js  
🗄️ **Database:** MongoDB  
🔐 **Auth:** Clerk  
☁️ **Media Storage:** Cloudinary  
💳 **Payment:** Stripe  
🌐 **Networking:** Axios  



---

## ✨ Features

✅ Dual Role Interfaces (Educators & Students)  
✅ Secure Authentication & Authorization (Clerk)  
✅ Video Streaming + Progress Tracking  
✅ Cloudinary Media Management  
✅ Course Enrollment + Secure Payments  
✅ Rating & Review System  
✅ Mobile Responsive Design  
✅ Search & Filter with Categories  
✅ Analytics Dashboard for Educators  
✅ Real-time Notifications
